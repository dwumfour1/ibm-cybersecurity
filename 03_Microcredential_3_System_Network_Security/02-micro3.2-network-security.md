# Micro 3.2: Network Security

## About This Learning Activity
In this module, you'll explore key concepts and best practices for **securing networks**. 

You will learn to identify threats, implement protections, and design networks with security in mind:

- Understand **network attacks**, how they work, and the impact they can have on systems and users.  
- Explore **network security devices** and learn how network administrators use them to prevent and mitigate attacks.  
- Examine **network security architecture** and how network design can proactively reduce vulnerabilities.  
- Learn about **network access controls** and why controlling access is vital for overall network security.

---

## Learning Objectives
By the end of this module, you should be able to:

1. Classify types of application and service attacks  
2. Classify types of wireless attacks  
3. Identify advantages of devices used to protect network security  
4. Design a secure network  
5. Differentiate between types of network access controls  

---
