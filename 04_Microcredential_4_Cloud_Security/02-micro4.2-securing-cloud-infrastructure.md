# Micro 4.2: Securing Cloud Infrastructure

## About This Learning Activity
In this module, you'll focus on **securing cloud infrastructure**, a critical aspect of modern cybersecurity.

You will learn to identify risks and apply security measures to protect cloud environments:

- Understand **cloud threats and vulnerabilities** that can compromise systems and data  
- Learn how to secure **cloud infrastructure and applications**  
- Apply **identity and access management (IAM)** to control and monitor access  
- Explore **recovery and backup strategies** to maintain availability  
- Gain hands-on understanding of creating secure virtual machines in cloud environments  

---

## Learning Objectives
By the end of this module, I am able to:

1. Apply identity and access management (IAM) in real-world scenarios  
2. Implement cloud infrastructure security and recovery plans  
3. Create a secure virtual machine using Microsoft Azure  
4. Apply security strategy principles to cloud applications  
5. Summarize key concepts for securing cloud data, infrastructure, and applications  

---

## Lessons

- Cloud Threats  
- Cloud Infrastructure Security  
- Cloud Application Security

## Reflection
This module strengthened my understanding of how to **secure cloud environments against real-world threats**. By applying IAM, securing infrastructure, and planning for recovery, I can ensure that cloud systems remain resilient and protected.  

Documenting these concepts helps me build the ability to **design and manage secure cloud solutions**, which is essential in modern cybersecurity roles.
