# Micro 4.3: Propose Cloud Services and Security Measures

## About This Learning Activity
In this capstone project, you will focus on **designing secure cloud solutions** for an organization transitioning to the cloud.

Using the case of Jasper Manufacturing, you will:

- Identify the most suitable **cloud computing framework and service model**  
- Recommend **security measures** to protect cloud systems, data, and infrastructure  
- Apply best practices to ensure **confidentiality, integrity, and availability (CIA)** in a cloud environment  

---

## Prerequisite Learning
This project builds on the knowledge and skills from:

- Micro 4.1: Cloud Computing and Virtualization  
- Micro 4.2: Securing Cloud Infrastructure  

---

## Learning Objectives
By the end of this module, I am able to:

1. Apply the appropriate cloud computing framework based on an organization’s needs  
2. Summarize key concepts for securing cloud data, infrastructure, and applications  

---

## Assessment Details
You will complete **two assessments** in this capstone project:

1. **Project Questions:**  
   Demonstrate your ability to secure cloud systems, data, applications, and infrastructure. A score of at least 80% is required, with multiple attempts allowed.

2. **10-Question Assessment:**  
   Test your understanding of the concepts applied in the project. A minimum score of 80% is required, with feedback and retry options available.

---

## Reflection
This capstone project helped me apply cloud security concepts in a **real-world scenario**. By selecting appropriate cloud models and implementing security controls, I strengthened my ability to design secure cloud environments.  

Documenting this process improves my understanding of how to **align cloud solutions with business needs while maintaining strong security practices**.
